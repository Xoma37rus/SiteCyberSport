# Руководство игрока

## 🚀 Начало работы

### Регистрация

1. Перейдите на главную страницу `http://localhost:8000`
2. Нажмите кнопку **«Присоединиться к НАМ»** или **«Регистрация»**
3. Заполните форму:
   - **Email** — действующий адрес электронной почты
   - **Имя пользователя** — уникальное имя (3-50 символов)
   - **Пароль** — минимум 6 символов
4. Нажмите **«Зарегистрироваться»**

> **Примечание:** В режиме разработки email подтверждается автоматически.

### Вход в аккаунт

1. Нажмите **«Войти»** в навигационной панели
2. Введите email и пароль
3. Нажмите **«Войти»**

После входа вы будете перенаправлены в **Личный кабинет (Dashboard)**.

## 🏆 Система рейтинга ELO

### Как работает рейтинг

EasyCyberPro использует рейтинговую систему, аналогичную FACEIT:

| Уровень | Диапазон ELO | Цвет бейджа |
|---------|--------------|-------------|
| Level 1 | 0-199 | Серый |
| Level 2 | 200-399 | Серый |
| Level 3 | 400-599 | Серый |
| Level 4 | 600-799 | Синий |
| Level 5 | 800-999 | Синий |
| Level 6 | 1000-1199 | Голубой |
| Level 7 | 1200-1399 | Голубой |
| Level 8 | 1400-1699 | Золотой |
| Level 9 | 1700-1999 | Золотой |
| Level 10 | 2000+ | Легендарный |

### Получение рейтинга

1. **Стартовый рейтинг:** 1000 ELO (Level 6)
2. **Изменение рейтинга** происходит после каждого матча
3. **Формула:** ΔR = K × (S - E)
   - K = 32 (базовый коэффициент)
   - S = фактический счёт (1=победа, 0=поражение)
   - E = ожидаемый счёт (на основе рейтинга соперника)

### Просмотр таблицы лидеров

1. Перейдите в меню **«Рейтинг»** → **«Таблица лидеров»**
2. Выберите дисциплину (Dota 2, CS2, Мир танков)
3. Просматривайте топ-100 игроков

Ваш личный рейтинг отображается в верхней части таблицы.

# Система рейтинга ELO

## 📐 Формула расчёта

### Базовая формула

```
ΔR = K × (S - E)
```

Где:
- **ΔR** — изменение рейтинга
- **K** — K-фактор (коэффициент изменения)
- **S** — фактический счёт (1=победа, 0=поражение, 0.5=ничья)
- **E** — ожидаемый счёт

### Ожидаемый счёт

```
E = 1 / (1 + 10^((R_соперника - R_игрока) / 400))
```

### K-факторы

| Опыт игрока | K-фактор |
|-------------|----------|
| < 10 игр | 48 (быстрое изменение) |
| 10-100 игр | 32 (базовый) |
| > 100 игр | 16 (стабильный) |

## 🎯 Примеры расчёта

### Пример 1: Победа над равным соперником

- Игрок A: 1200 ELO
- Игрок B: 1200 ELO
- Результат: A побеждает

```
E = 1 / (1 + 10^((1200 - 1200) / 400)) = 0.5
ΔR = 32 × (1 - 0.5) = +16
Новый рейтинг A: 1200 + 16 = 1216
```

### Пример 2: Поражение от более сильного

- Игрок A: 1000 ELO
- Игрок B: 1400 ELO
- Результат: A проигрывает

```
E = 1 / (1 + 10^((1400 - 1000) / 400)) = 0.09
ΔR = 32 × (0 - 0.09) = -3
Новый рейтинг A: 1000 - 3 = 997
```

### Пример 3: Победа над более сильным (бонус)

- Игрок A: 1000 ELO
- Игрок B: 1300 ELO
- Результат: A побеждает

```
E = 1 / (1 + 10^((1300 - 1000) / 400)) = 0.15
ΔR = 32 × (1 - 0.15) = +27
Бонус за победу над сильным: +5
Итого: +32
Новый рейтинг A: 1000 + 32 = 1032
```

# Частые вопросы (FAQ)

## 🔑 Аккаунт

1. На странице входа нажмите **«Забыли пароль?»**
2. Введите email

5. Введите новый пароль

### Как изменить email?

Напишите в поддержку через форму обратной связи.

---

## 🏆 Турниры

### Что делать, если заявка не подтверждается?

Свяжитесь с организатором турнира через раздел **«Контакты»**.

### Можно ли отменить заявку?

Да, на странице турнира нажмите **«Отменить регистрацию»**.

### Как формируется турнирная сетка?

После завершения регистрации система автоматически создаёт сетку Single Elimination.

---

## 📊 Рейтинг

### Как быстро меняется рейтинг?

После каждого матча. Изменение зависит от:
- Количества сыгранных игр
- Разницы рейтинга с соперником
- Результата матча

### Что такое винрейт (Win Rate)?

Процент побед от общего количества игр:
```
Win Rate = (Победы / Игры) × 100%
```

### Можно ли потерять уровень?

Да, если ваш ELO опустится ниже порога текущего уровня.

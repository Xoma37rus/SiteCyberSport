```python
ELO_CHANGE = 32  # K-factor
EXPECTED_SCORE = 1 / (1 + 10 ** ((rating_b - rating_a) / 400))

```

#### 1.4 Автоматический матчмейкинг

**Задачи:**
- [ ] Создать модель `MatchmakingQueue` (user_id, discipline_id, elo, timestamp)
- [ ] Реализовать алгоритм подбора игроков (по ELO ±200)
- [ ] Таймаут ожидания (max 5 минут)
- [ ] Уведомление о найденном матче
- [ ] Автоматическое создание матча в БД

**Алгоритм:**
```python
def find_match(queue, user):
    candidates = queue.filter(
        discipline_id=user.discipline_id,
        elo BETWEEN user.elo - 200 AND user.elo + 200
    ).order_by(timestamp)
    return candidates[:5]  # 5v5
```

**Файлы:**
- `models.py` — MatchmakingQueue
- `matchmaking.py` — новый модуль
- `main.py` — background task для matchmaking
